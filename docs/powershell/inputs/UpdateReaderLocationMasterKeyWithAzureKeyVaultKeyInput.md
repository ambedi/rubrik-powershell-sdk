### UpdateReaderLocationMasterKeyWithAzureKeyVaultKeyInput
Input for updating reader location's master key with Azure Key Vault key.

- locationId: System.String
  - Field for specifying location identifier of the archival location.
- newAzureKeyIdentifier: AzureKeyVaultKeyIdentifierWithoutKeyVersionInput
  - Identifier for Azure Key Vault Key to be used for updating reader location's master key.
