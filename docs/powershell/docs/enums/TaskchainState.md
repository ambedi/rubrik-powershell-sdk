### TaskchainState
Job instance state.

- READY - Job instance is waiting to be executed.
- RUNNING - Job instance is running.
- SUCCEEDED - Job instance succeeded.
- UNDOING - Job instance has hit an error and is undoing.
- FAILED - Job instance failed.
- CANCELING - Job instance is canceling.
- CANCELED - Job instance canceled.
- UNKNOWN - Job instance is in an unknown state.
