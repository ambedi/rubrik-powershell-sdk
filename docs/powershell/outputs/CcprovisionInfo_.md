### CcprovisionInfo
Metadata for a single cluster job.

- vendor: CcpVendorType
  - Cloud vendor provider.
- jobStatus: CcpJobStatus
  - Current job status.
- progress: System.Int32
  - Progress of job in percent.
- jobType: CcpJobType
  - Type of running job.
- marshaledConfig: System.String
  - Cloud cluster configuration.
