### MetadataComponentChange
A Salesforce metadata component change.

- action: AppItemActionType
  - Action performed on the metadata component.
- componentName: System.String
  - Type of the metadata component.
- lastModifiedBy: System.String
  - User who last modified the metadata component.
- lineChangeSummary: LineChangeSummary
  - Summary of the line changes in the metadata component.
- componentId: System.String
  - ID of the metadata component.
- appItemTypeToken: System.String
  - Token specifying the app item type.
