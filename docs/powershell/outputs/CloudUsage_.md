### CloudUsage
Cloud usage.

- ccesUsage: CcesUsage
  - Cloud Cluster - Elastic Storage usage data.
- cnpUsage: CnpUsage
  - Cloud native applications usage data.
- frontEndBytesConsumed: System.Int64
  - Total cloud front-end capacity consumed, in bytes.
- timestamp: DateTime
  - Timestamp of the cloud usage data.
