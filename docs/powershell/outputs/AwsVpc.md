### AwsVpc
A Virtual Private Cloud (VPC) in AWS realm.

- id: System.String
  - ID for the VPC.
- name: System.String
  - Name of the VPC.
- subnets: list of AwsSubnets
  - List of subnets associated with the VPC.
- securityGroups: list of AwsSecurityGroups
  - List of security groups associated with the VPC.
