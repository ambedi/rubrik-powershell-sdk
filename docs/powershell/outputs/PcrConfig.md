### PcrConfig
Details of Private Container Registry for Exocompute account

- hasPcr: System.Boolean
  - Whether the account has Private Container Registry.
- pcrUrl: System.String
  - URL of Private Container Registry.
- pcrLatestApprovedBundleVersion: System.String
  - Latest approved bundle version in Private Container Registry.
- pcrImagePullAwsNativeId: System.String
  - AWS native account ID authorized to pull images from an Amazon Elastic Container Registry (ECR) hosted by Rubrik.
